異世界シリーズを追う入口

『だから俺は異世界で春を売る。』シリーズ3冊を中心に、多種族と異世界設定を追いやすくした特集です。

異世界男娼もの多種族
COLLECTION Curated path

この特集の意図

単巻としても読めますが、1巻から順に追うと世界観と主人公の立ち位置が分かりやすいシリーズ導線です。

見方のポイント

  • まず1巻で世界観と主人公の位置をつかむ
  • 2巻で体型と種族の幅を広げる
  • 3巻でライオン中心の回へ進み、好みの相手傾向を探す
