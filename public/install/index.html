起動ガイド

イチメAI（日本語AIコーディング入門）— Windows β

購入後のダウンロードから初回起動までを、実際の導線に合わせて短くまとめました。

まず、購入後はどうなる？

  1. 決済が完了すると購入完了ページ（/thanks）が表示されます。
  2. 「ダウンロード」を押すと、/dl?token=... のリンクからインストーラ（.exe）が保存されます。
  3. インストール後、アプリを起動します。
  4. 初回はアプリ内の「ログイン」から ChatGPT公式ログイン画面 で認証します（Plus/Pro/Business/Edu/Enterprise）。
  5. ChatGPT Go は対象外（使用不可）です。

30秒で始める（最短）

  1. /thanks で「ダウンロード」を押す
  2. ダウンロードした .exe をダブルクリック
  3. インストール後にアプリを起動 → 「ログイン」
  4. ログイン完了後に「状態更新」を押す
  5. 「新しく作る」を選び、やりたいことを1文で入力

1) ダウンロード

  • /thanks で「ダウンロード」を押します。
  • 画面が進まない場合は「再確認」を押してください。

2) インストール

  • ダウンロードした .exe を実行します。
  • SmartScreenが出た場合は「詳細情報」→「実行」で進めます。

3) 初回起動とログイン

  • アプリを起動して「ログイン」を押します。
  • ブラウザに表示される ChatGPT公式ログイン画面 で認証します。
  • 利用には Plus/Pro/Business/Edu/Enterprise が必要です。
  • ChatGPT Go は対象外（使用不可）です。
  • APIキー方式は非対応（入力欄なし）。
  • 完了後に「状態更新」で先へ進みます。

ログイン画面の例は こちら を参照してください。

4) 最初の成功体験

  • 「新しく作る」を選びます。
  • 入力欄にやりたいことを短く書きます（1文でOK）。

よくあるつまずき

ダウンロードできない / 何も起きない

/thanks の「再確認」を押してから、もう一度「ダウンロード」を試してください。

「download not found」などが出る

セッション情報が読み取れない可能性があります。/thanks に戻り、再確認を押してからやり直してください。

起動しない / すぐ落ちる

セキュリティソフトが実行を止めている場合があります。例外設定や別PCでの確認をお試しください。

ログインがうまくいかない

企業ネットワークなどで外部通信が制限されている可能性があります。別の回線でお試しください。

困ったとき（サポート）

  • どの画面で止まったか（スクショOK）
  • 表示されたメッセージ（コピペOK）
  • Windowsのバージョン
