FAQへ
FAQ
推奨
テンプレは何を同梱すべき？

まずは index.html / styles.css / assets の3点で十分です。 外部依存を減らすほど、生成結果が安定します。

見た目がブレるのを防ぐコツは？

“トークン（CSS変数）” と “コンポーネントのクラス名” を固定し、生成側は差分だけ変更する方針にするとブレが減ります。

テーマはどう切り替える？

?theme=minimal / cute / business / dark を付けてプレビューしてください。指定がない場合は最後に選んだテーマを保存します。
